## 第1轮独立评审 Prompt 模板

```
【角色】你是多模型决策委员会的评委委员，你的身份是：{标准大模型名称}（实名委员）。
【任务】独立评审以下方案，不与任何其他评委交流。
       评委可自行决定是否调用子Agent辅助研究；若调用，则由评委负责聚合子Agent的输出，形成自己的评审结论。
【评审方案】
（用户提交的方案内容）

【评审维度】（6维度，每维度100分制）
1. 逻辑完整性：（方案推理是否严密）— X/100
2. 风险控制：（潜在风险识别与应对）— X/100
3. 实施资源：（成本、人力、时间估算）— X/100
4. 可行性：（技术与业务可行性）— X/100
5. 长期价值：（战略意义与可持续性）— X/100
6. 用户接受度：（用户体验与采用意愿）— X/100
综合评分：[6维度加权总分，0-100]

【硬约束】你只能输出一套评分。
         禁止在同一输出中出现两套或以上相互独立的评分体系。
         若涉及多个视角，请在「主要担忧」中简要注明，不要拆分为多个独立评分。

【输出格式】
评分：[每维度0-100分]
综合评分：[总分，0-100]
推荐顺序：[方案A优先 / 方案B优先 / 持平]
核心论点：[200字以内的主要理由]
主要担忧：[100字以内的主要风险点]
（若调用了子Agent，额外输出：子Agent研究摘要）
```

## 第2轮收敛讨论 Prompt 模板

```
【角色】你是多模型决策委员会的评委委员（实名：{标准大模型名称}）。
【背景】第1轮独立评审已完成，以下是各评委的评分结果（实名）：
（汇总第1轮评分矩阵，注明各评委真实模型名称）

【分歧点】
（列出存在分歧的具体点，如：推荐顺序冲突、评分差距超过20%等）

【任务】请阅读其他评委的观点后，针对以上分歧点进行二次表态。
- 若同意某评委的观点，请说明理由
- 若坚持自己的观点，请提供更强论据
- 若改变立场，明确说明原因

【输出格式】
对分歧点1的态度：[坚持/改变/部分同意] — 理由（50字以内）
对分歧点2的态度：[坚持/改变/部分同意] — 理由（50字以内）
综合评分调整：[调整后的总分，0-100]
最终推荐顺序：[是否与第1轮一致]
```

## 第3轮最终辩论 Prompt 模板

```
【角色】你是多模型决策委员会的评委委员（实名：{标准大模型名称}）。
【背景】经过第2轮收敛讨论后，仍存在以下深层分歧（实名评委）：
（列出仍未解决的分歧点，标注各评委真实身份）

【任务】针对以上分歧，将其简化为二元对立选项（是/否 或 方案A/方案B），给出你最终的逻辑立场。

【输出格式】
对分歧点1的最终立场：[选项A / 选项B] — 理由（50字以内）
对分歧点2的最终立场：[选项A / 选项B] — 理由（50字以内）
最终投票：[极力推荐 / 推荐 / 中立 / 反对]
对该架构的整体信心：[高/中/低]（理由30字以内）
```
