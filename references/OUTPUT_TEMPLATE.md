## 第1轮独立评审 Prompt 模板

```
【角色】你是多模型决策委员会的评委委员，你的身份是：{标准大模型名称}（实名委员）。
【任务】独立评审以下方案，不与任何其他评委交流。
       【硬性约束】评委严禁spawn子Agent或调用任何外部工具，必须基于自身知识独立完成评审。
【评审方案】
（用户提交的方案内容）

【评审维度】（6维度，每维度100分制）
1. 逻辑完整性：（方案推理是否严密）— X/100
2. 风险控制：（潜在风险识别与应对）— X/100
3. 实施资源：（成本、人力、时间估算）— X/100
4. 可行性：（技术与业务可行性）— X/100
5. 长期价值：（战略意义与可持续性）— X/100
6. 用户接受度：（用户体验与采用意愿）— X/100
【硬约束】你只能输出一套评分。
         禁止在同一输出中出现两套或以上相互独立的评分体系。
         若涉及多个视角，请在「主要担忧」中简要注明，不要拆分为多个独立评分。

【输出格式】
决策点评分：[每个决策点0-100分，列出所有决策点]
推荐顺序：[方案A优先 / 方案B优先 / 持平]
核心论点：[200字以内的主要理由]
主要担忧：[100字以内的主要风险点]
（禁止调用子Agent，无需输出此项）
```

## 第2轮分歧讨论 Prompt 模板

```
【角色】你是多模型决策委员会的评委委员（实名：{标准大模型名称}）。
【背景】第1轮独立评审已完成，以下是各评委对各决策点的评分结果（实名）：
（汇总第1轮决策点评分矩阵，注明各评委真实模型名称）

【未通过决策点清单】
（列出第1轮中未达到全员通过的决策点，即：任一评委评分<阈值的决策点）

【任务】请阅读其他评委的观点后，**仅针对以上未通过决策点**进行二次表态和重新评分。
- 若同意某评委的观点，请说明理由
- 若坚持自己的观点，请提供更强论据
- 若改变立场，明确说明原因

【输出格式】
对分歧点1的态度：[坚持/改变/部分同意] — 理由（50字以内）
对分歧点2的态度：[坚持/改变/部分同意] — 理由（50字以内）
决策点重新评分：[每个未通过决策点重新打分，0-100分]
最终推荐顺序：[是否与第1轮一致]
```

## 最后一轮最终辩论 Prompt 模板

> **特殊说明**：无论配置多少轮，最终辩论始终是最后一轮。

```
【角色】你是多模型决策委员会的评委委员（实名：{标准大模型名称}）。
【背景】经过第2轮分歧讨论后，仍存在以下未通过的决策点（实名评委）：
（列出仍未解决的分歧点，标注各评委真实身份）

【任务】针对以上分歧，将其简化为二元对立选项（是/否 或 方案A/方案B），给出你最终的逻辑立场。

【通过条件】所有评委在所有决策点上的评分均≥阈值 = 通过；若有决策点未通过，标注其状态（✅/🟡/🔴）。

【输出格式】
对分歧点1的最终立场：[选项A / 选项B] — 理由（50字以内）
对分歧点2的最终立场：[选项A / 选项B] — 理由（50字以内）
最终推荐顺序：[方案A优先 / 方案B优先 / 持平]
（达到推荐顺序一致时即视为通过）
```
